# JR's Home Cleaning — Static Site

Pure HTML, CSS, and JavaScript. No build step, no dependencies.

## Run locally

Just open `index.html` in your browser, or serve the folder:

```bash
# any of these works
python3 -m http.server 8080
npx serve .
```

Then open http://localhost:8080.

## Files

- `index.html` — markup
- `styles.css` — all styling
- `script.js` — calculator logic, carousel, WhatsApp booking
- `assets/` — images
